# Financial Risk Analysis using K-Means Clustering

This project demonstrates **financial risk analysis** using Python and K-Means clustering. The project:

- Loads historical stock price data
- Calculates daily returns and risk metrics (volatility and mean returns)
- Applies K-Means clustering to classify stocks into risk groups
- Visualizes the clusters for better investment insights

**Technologies Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

**Dataset:** Historical stock prices (can be downloaded from Yahoo Finance)
